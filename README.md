🏏 Box Cricket Website

A modern Box Cricket booking and tournament management website built using PHP, MySQL, HTML, CSS, and Bootstrap.

✨ Features

* User & Admin Login System
* Ground & Tournament Booking
* Seat Selection System
* Online Payment Support
* User Dashboard
* Admin Management Panel
* Contact & Feedback System
* Forgot Password with OTP Verification

## 🛠️ Technologies Used

* HTML
* CSS
* Bootstrap
* PHP
* MySQL

This project helps users easily book cricket grounds, join matches, and manage bookings online.
